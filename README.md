## Mise en ligne GitHub Pages
1. Upload `index.html`, `styles.css`, `script.js` et le dossier `assets` dans le repo GitHub.
2. Va dans Settings > Pages.
3. Source : Deploy from a branch.
4. Branch : main, folder : /root.
5. Save.
